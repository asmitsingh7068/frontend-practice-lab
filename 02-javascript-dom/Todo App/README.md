# 📝 Todo App

A simple, responsive, and interactive Todo App built using **HTML**, **CSS**, and **JavaScript (DOM)**. This project helps users manage daily tasks with features like adding, editing, completing, deleting, and saving tasks using Local Storage.

---

## 🚀 Features

- ✅ Add new tasks
- ✏️ Edit existing tasks
- ✔️ Mark tasks as completed
- 🗑️ Delete tasks
- ⌨️ Add tasks using the Enter key
- 💾 Local Storage support (tasks remain after page refresh)
- 📊 Live task counter
- 📱 Responsive design
- 🎨 Clean and modern UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- DOM Manipulation
- Local Storage API
