# 🖼️ Image Slider

A responsive and interactive **Image Slider (Carousel)** built using **HTML, CSS, and JavaScript**. This project demonstrates DOM manipulation, event handling, automatic image transitions, and responsive UI design.

---

## ✨ Features

- ✅ Previous & Next Navigation
- ✅ Automatic Image Sliding
- ✅ Navigation Dots
- ✅ Infinite Loop
- ✅ Smooth Slide Animation
- ✅ Responsive Design
- ✅ Modern UI
- ✅ Easy to Customize
