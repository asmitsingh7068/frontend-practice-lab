# 💳 Pricing Section

A responsive and modern **Pricing Section** built using **HTML5** and **CSS3**. This project showcases different pricing plans with clean card layouts, hover effects, customer reviews, FAQ section, and a professional footer.

---

## 🚀 Features

- 🎯 Hero Section
- 💳 Four Pricing Cards
- ⭐ Featured "Most Popular" Plan
- 🎨 Clean and Modern UI
- 🖱️ Hover Effects
- 📱 Beginner-Friendly Layout
- ✅ Why Choose Us Section
- 💬 Customer Reviews
- ❓ FAQ Section
- 📌 Professional Footer

---

## 🛠️ Technologies Used

- HTML5
- CSS3
