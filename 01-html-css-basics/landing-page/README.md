# Responsive Landing Page

A modern, clean, and responsive landing page built using **HTML, CSS, and JavaScript** as part of my **Frontend Practice Lab** repository.

## Objective
This project was created to practice building a complete landing page layout with reusable UI sections and responsive design principles.

## Features
- Responsive Navbar
- Hero Section with CTA buttons
- Feature Cards Section
- How It Works / Workflow Section
- Statistics Section
- Testimonials Section
- Pricing Section Preview
- FAQ Section
- Footer Section

## Tech Used
- HTML5
- CSS3
- JavaScript

## Concepts Practiced
- Semantic HTML
- Flexbox
- CSS Grid
- Responsive Design
- Reusable UI Patterns
- Layout Structuring
- Typography and Spacing
