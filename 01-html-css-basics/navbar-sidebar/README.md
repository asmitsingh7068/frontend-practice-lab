# Navbar Sidebar | HTML & CSS

A simple and responsive **Navbar + Sidebar Dashboard UI** built using **HTML5** and **CSS3**.

## 🚀 Features

- Responsive Navbar
- Sidebar Navigation
- Dashboard Layout
- Welcome Banner
- Dashboard Cards
- Recent Orders Table
- Clean UI Design
- Hover Effects
- Modern Color Theme

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
