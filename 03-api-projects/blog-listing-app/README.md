# 📝 Blog Listing App

A responsive **Blog Listing Application** built using **HTML, CSS, and JavaScript**. The application loads blog data from a local JSON file using the **Fetch API** and allows users to search and filter blogs by category.

# 📡 Data Source

The project uses a local JSON file.

```text
blogs.json
```

Each blog contains:

- Blog Title
- Category
- Author
- Publish Date
- Read Time
- Blog Image
- Description
- Full Content

# 🎯 Key Functionalities

- Fetches blog data from `blogs.json`
- Dynamically creates blog cards
- Search blogs by title or description
- Filter blogs by category
- Displays loading spinner while fetching data
- Shows error message if data fails to load
- Responsive layout for mobile, tablet, and desktop
