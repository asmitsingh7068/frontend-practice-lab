# 💼 Job Board UI

A responsive **Job Board UI** built using **HTML, CSS, and JavaScript**. This project displays job listings from a local JSON file and allows users to search and filter jobs dynamically.

# 📡 Data Source

The project uses a local JSON file.

```text
jobs.json
```

The JSON file contains job information such as:

- Job Title
- Company Name
- Location
- Job Type
- Salary
- Company Logo
