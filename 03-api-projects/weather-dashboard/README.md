# 🌦 Weather Dashboard

A simple and responsive Weather Dashboard built using **HTML**, **CSS**, and **JavaScript**. This project fetches real-time weather information from the OpenWeather API and displays the current weather conditions for any city.

## 🔑 API Setup

1. Visit **https://openweathermap.org/api**
2. Create a free account.
3. Generate your API Key.
4. Open `script.js`.
5. Replace:

```javascript
const apiKey = "asknkdnvenor";
```

with your own API key.

Example:

```javascript
const apiKey = "abc123xyz456";
```

## ▶ How to Run

1. Clone this repository.

```bash
git clone https://github.com/asmitsingh7068/frontend-practice-lab.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

4. Enter a city name.

5. Click **Search**.

6. View the live weather information.
