# Frontend Practice Lab

A curated collection of frontend assignments and mini-projects built to strengthen my frontend development skills in HTML, CSS, JavaScript, responsive design, DOM manipulation, reusable UI components, and real-world interface development.

## Repository Goals
This repository is created to:
- Practice frontend fundamentals
- Build real-world UI assignments
- Improve JavaScript DOM skills
- Work on API-based mini projects
- Learn reusable component development
- Maintain clean project structure and documentation
