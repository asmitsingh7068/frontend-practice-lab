# 📋 Form Component

A modern, responsive, and reusable Registration Form built using **HTML**, **CSS**, and **JavaScript**. This project demonstrates professional form design, client-side validation, and clean UI practices.

---

## 📸 Preview

Responsive Registration Form with:

- Full Name
- Email
- Password
- Confirm Password
- Phone Number
- Date of Birth
- Gender Selection
- Country Dropdown
- Address
- Terms & Conditions
- Register Button

---

## ✨ Features

- ✅ Modern Glassmorphism UI
- ✅ Fully Responsive Design
- ✅ Clean Form Layout
- ✅ Required Field Validation
- ✅ Password Confirmation Validation
- ✅ Show / Hide Password
- ✅ Gender Selection
- ✅ Country Dropdown
- ✅ Terms & Conditions Validation
