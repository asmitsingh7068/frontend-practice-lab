# 🎨 Premium Modal Component

A modern and responsive Modal (Popup) component built using **HTML**, **CSS**, and **JavaScript**. This project demonstrates DOM manipulation, event handling, animations, and responsive UI design.

---

## 🚀 Features

- ✅ Modern Glassmorphism UI
- ✅ Responsive Design
- ✅ Smooth Open Animation
- ✅ Open Modal Button
- ✅ Close Button (×)
- ✅ Cancel Button
- ✅ Continue Button
- ✅ Close on Outside Click
- ✅ Close with ESC Key
- ✅ Hover Effects
- ✅ Clean and Reusable Code

---

## 🛠️ Tech Stack

- HTML5

- JavaScript (ES6)

## 🎮 Functionality

- Click **Open Modal** to display the popup.
- Click **Cancel** or **×** to close the modal.
- Click **Continue** to perform an action.
- Click outside the modal to close it.
- Press the **ESC** key to close the modal.
