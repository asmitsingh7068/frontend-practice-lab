# 🃏 Reusable Card Library

A modern and responsive **Reusable Card Library** built using **HTML**, **CSS**, and **JavaScript**. This project demonstrates how to create reusable UI card components that can be dynamically rendered using JavaScript.

---

## 📸 Preview

The project includes multiple reusable card components such as:

- 🛍️ Product Card
- 👤 Profile Card
- 📚 Course Card
- 📝 Blog Card
- 👥 Team Card
- 💎 Pricing Card

## 🚀 How to Run

1. Clone the repository.

```bash
git clone https://github.com/asmitsingh7068/frontend-practice-lab.git
```

2. Open the project folder.

3. Navigate to:

```text
09-reusable-card-library
```

4. Open `index.html` in your browser.

No additional installation is required.
